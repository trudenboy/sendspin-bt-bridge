## На что идут донаты

- **🛠️ Тестовое железо.** Каждая новая модель Bluetooth-колонки — IKEA, Sony, JBL, Yandex, noname-бренды — имеет свои особенности BlueZ, которые можно отладить только на реальном устройстве.
- **🌐 Инфраструктура** — домен, хостинг документации, CI-раннеры, периодически VPS для тестового стенда.
- **⏱️ Время** на отлов регрессий BlueZ и Linux Bluetooth-стека — там основная неблагодарная работа по поддержке.

## FAQ

### Что я получаю за донат?

Честно: ничего эксклюзивного. Все функции доступны всем, исходный код открытый, релизы выходят на GitHub в тот же день. Вы поддерживаете проект, а не покупаете доступ.

### Можно ли получить налоговый вычет?

Нет — это личный open-source проект, не зарегистрированная НКО. Донаты — это добровольный взнос, а не благотворительность в налоговом смысле.

### Моей страны нет в списке поддерживаемых Liberapay / Boosty / NOWPayments. Что делать?

Прямые крипто-адреса работают из любой точки мира. Если крипта тоже не вариант — напишите в [GitHub Issues](https://github.com/trudenboy/sendspin-bt-bridge/issues), для крупных разовых донатов можно организовать SWIFT/Wise-перевод.

### Почему нет Stripe / PayPal / GitHub Sponsors / Buy Me a Coffee?

Все эти сервисы зависят от Stripe для выплат, а Stripe не поддерживает Казахстан и Россию (где живёт автор). Обход потребовал бы регистрации US LLC или UK Ltd, что добавляет compliance-нагрузку, несоразмерную с объёмом opensource-донатов.

### Можно ли посмотреть, на что потрачены донаты?

Периодически отчитываюсь в [журнале изменений](https://github.com/trudenboy/sendspin-bt-bridge/blob/main/CHANGELOG.md) и release notes — например, когда новое тестовое устройство привело к фиксу совместимости.
