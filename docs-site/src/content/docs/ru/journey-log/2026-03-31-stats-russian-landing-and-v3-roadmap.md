День после запуска лендинга сместил фокус с фич на видимость проекта и долгосрочное планирование. Дашборд статистики, русский лендинг, интеграция аналитики и комплексная дорожная карта v3 — всё за один день, плюс CI для бета-ветки и v2.52.1 с исправлениями LXC-обновлений.

## Что выпущено

### Дашборд статистики

Документационный сайт на Astro Starlight получил полную страницу статистики проекта с живыми данными GitHub API:

- **SVG area chart** трафика (просмотры, клоны) — заменил более простой bar-row дизайн через три итерации, потому что CSS-переменные не работают в SVG-контексте
- **Stacked bar chart** загрузок релизов — с реальными данными GHCR container pull через packages API
- **Секция активности разработки** — скорость кода, статистика AI-коллабораций (Copilot vs Claude Code), разбивка по языкам
- **Issues & PRs** — соотношение open/closed, среднее время закрытия
- **CI workflows** — процент успеха, средняя длительность, последние запуски
- **Тайлы-индикаторы** — container pulls, звёзды, форки, статус CI

### Русский лендинг

Вместо runtime-виджета Google Translate (который мерцает и переформатирует страницу) создан статический перевод через DeepL API:

- Полный статический перевод с ручной правкой технических терминов
- Переключатель языков ведёт на статические страницы, а не запускает рантайм-перевод
- Исправлено переполнение заголовков на мобильных 375 px через `<wbr>` и уменьшенный размер шрифта

### SEO и аналитика

- **Structured data**: FAQ Schema, Product markup
- **Социальные превью**: оптимизированные OG-изображения (~200 KB → ~50 KB)
- **Аналитика**: GA4, Яндекс.Метрика, Microsoft Clarity
- **Search Console**: верификация для Google и Яндекс, robots.txt, sitemap

### Дорожная карта v3.0+ (PR #118)

Опубликован `ROADMAP_V3.md` — план трансформации из `sendspin-bt-bridge` в `sendspin-audio-bridge`:

- **Track A**: Абстракция бэкендов — единый `AudioBackend` интерфейс для Bluetooth, локального аудио, сетевых протоколов
- **Track B**: Федерация мультибриджей
- **Track C**: HA/AI автоматизация — нативные сущности HA, маршрутизация по присутствию
- **Track D**: CLI для управления — `sendspin-cli`

### CI бета-канала

CI расширен для поддержки ветки `beta` для разработки v3.0. Первые 14 бета-релизов выпущены (v3.0.0-beta.1 — beta.14), Vue 3 фронтенд достиг полного паритета с легаси UI — 108 фич, 489 Vue-тестов.

### v2.52.1

Исправление LXC-обновлений: удалён `demo/` из списка синхронизации `upgrade.sh`, добавлены проверки существования директорий.

## Почему это важно

Дашборд статистики и русский лендинг — инвестиции в сообщество. Дорожная карта v3 определяет архитектурное направление за пределами Bluetooth. CI для бета-ветки — инфраструктура для миграции Vue 3.
