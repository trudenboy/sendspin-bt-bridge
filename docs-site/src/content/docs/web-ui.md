Веб-интерфейс доступен на порту **8080** (Docker/LXC) или через **HA Ingress** (аддон).

## Панель управления

![Панель управления Sendspin Bluetooth Bridge с 6 карточками устройств](/sendspin-bt-bridge/screenshots/screenshot-dashboard-full.png)

### Групповые элементы управления

В верхней части страницы расположена панель групповых операций:
- **Флажок All** — включить/отключить все устройства
- **Слайдер Vol** — регулировка громкости всех активных устройств одновременно
- **🔈 Mute All** — отключить звук на всех устройствах
- **▮▮ Pause All** — поставить на паузу все воспроизводящие плееры

### Карточка устройства

Каждое Bluetooth-устройство отображается в отдельной карточке:

![Карточка устройства ENEBY20 со статусом Bluetooth, сервера, громкостью и синхронизацией](/sendspin-bt-bridge/screenshots/screenshot-device-card.png)

| Секция | Содержание |
|---|---|
| **Заголовок** | Название, MAC-адрес, WebSocket URL |
| **Bluetooth** | Статус подключения, время подключения, адаптер и его MAC |
| **Server** | Статус подключения к MA серверу, IP:порт |
| **Playback** | ▶ Playing / ▮▮ Paused / No Sink, кнопка паузы/воспроизведения, формат аудио |
| **Volume** | Слайдер громкости, кнопка mute |
| **Sync** | Статус синхронизации, количество переякорений, текущая задержка |
| **Управление** | Кнопки Reconnect, Re-pair, Release/Reclaim |
| **Track** | Исполнитель — Название трека |

### Кнопки управления устройством

| Кнопка | Действие |
|---|---|
| **🔄 Reconnect** | Принудительно переподключить Bluetooth |
| **🔗 Re-pair** | Повторный паринг (~25 сек, переведите устройство в режим паринга) |
| **🔓 Release** | Освободить устройство — мост перестаёт управлять подключением (устройство остаётся в системе) |
| **🔒 Reclaim** | Вернуть управление устройством |

<Aside type="tip">
  **Release** полезен, если вы хотите временно подключить колонку к другому источнику (телефону, ПК) не останавливая мост.
</Aside>

## Конфигурация

Раскрывающийся раздел **⚙️ Configuration** позволяет изменить настройки без редактирования файла:

![Раздел конфигурации с полями настроек и таблицей устройств](/sendspin-bt-bridge/screenshots/screenshot-config.png)

Раскрывающийся раздел **⚙️ Configuration** позволяет изменить настройки без редактирования файла:

- **MA Server / Port** — адрес и порт Music Assistant
- **Bridge Name** — суффикс к именам плееров
- **PulseAudio latency** — задержка в мс
- **Prefer SBC codec** — принудительный SBC
- **Timezone** — временная зона
- **Bluetooth Adapters** — список адаптеров с кнопками Refresh и Add
- **Bluetooth Devices** — таблица устройств с кнопками Add Device и Scan

Кнопки сохранения:
- **Save Configuration** — сохранить без перезапуска
- **Save & Restart** — сохранить и перезапустить сервис

## Диагностика

Раздел **Diagnostics** отображает:
- Список Bluetooth-адаптеров и их состояние
- Аудио синки (PulseAudio/PipeWire sinks)
- D-Bus статус

## Журналы

Раздел **📋 Logs** выводит последние записи из лога приложения в реальном времени.

## Тема оформления

Интерфейс автоматически применяет тему:
- Через HA Ingress — читает тему Home Assistant через `postMessage` API
- Напрямую — следует системной теме браузера (`prefers-color-scheme`)

## API

Все данные веб-интерфейса доступны через REST API. Подробнее — в разделе [API Reference](/sendspin-bt-bridge/api/).
